## 📢 동네방송 안내

이곳은 **대한민국 최고 생일남 병준이**를 축하하기 위해 만든 온라인 특설무대입니다.

세련됨은 잠시 내려놓고, 2000년대 개인 홈페이지와 동네 잔치 무대의 뜨거운 정성을 한 화면에 담았습니다.

## 🎤 오늘의 무대 순서

- 🌈 무지개로 번쩍이는 초대형 생일 현수막
- ✨ 사방팔방 떠다니는 별, 하트, 풍선
- 🎁 누를 때마다 바뀌는 주옥같은 병준이 맞춤 생일 덕담 20선
- 🎉 화면을 뒤흔드는 이모지 폭죽 48발
- 🚨 암전 후 등장하는 초대형 최종 축하 무대
- 💖 촌스럽지만 진심만큼은 최고급인 우정

## 🎹 우리 집에서도 풍악 울리는 법

준비물은 Node.js `22.13.0` 이상입니다.

```bash
npm install
npm run dev
```

잔치가 제대로 열리는지 확인하려면:

```bash
npm run build
```
